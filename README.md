# ICA Components

Component library for ICA Sweden - connects to Leigher personalization platform.

## Components

### Product Card
`product-card.html` - Reusable product card with Open Food Facts API integration.

**Data fields:**
- `title` - Product name
- `price` - Price in SEK
- `size` - Weight/volume
- `unitPrice` - Price per kg/l
- `origin` - Country of origin
- `image` - Product image URL

**Data source:** [Open Food Facts API](https://world.openfoodfacts.org/api/v2/search)

## Usage with Leigher

Connect this repo via GitHub integration in Leigher dashboard.
